# Default Memory Directory

## Description

This is the default memd memory directory.

Use it for general memory unless the user creates more specific memory directories.

## Use When

Use this directory when the task depends on remembered context, preferences, decisions, rejected ideas, reusable procedures, examples, ongoing work, or open questions.

## Do Not Store

- Secrets, credentials, tokens, passwords, private keys, or recovery codes.
- Information that clearly belongs in another memory directory.
- Sensitive personal, work, team, or public-facing information unless the user explicitly wants it stored here.

## Write Rules

- Update this directory only when the information should help future agents.
- Prefer editing existing pages.
- Create new pages only when the idea has durable independent meaning.
- Link related pages.
- Ask before storing sensitive or inferred preferences.
- Keep the wiki flat until structure is useful.

## Isolation Rules

This memory directory is isolated by default.

Do not copy information into or out of another memory directory unless the user explicitly asks or the relevant directory rules allow it.
