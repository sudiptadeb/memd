How it works

memd exposes selected local memory directories through scoped connector URLs.

Get started

  1. Shared memory is already waiting for you.
  2. Add a directory. A folder or git repository where your memory files live.
  3. Create a connector. A scoped URL your agent uses to reach the directories you pick — choose which ones, and whether the agent may write.
  4. Paste the connector URL into your agent. In Claude Code: claude mcp add --transport http memd "<connector URL>". Any other MCP client takes the same URL in its server settings. From the next conversation on, the agent loads and updates memory on its own.
